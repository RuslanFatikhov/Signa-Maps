# Changelog

## 1.3

### Fixes
- Fixed an issue where locations wouldn’t open in Yandex Maps.
- Fixed an issue where locations wouldn’t open in Organic Maps.
- Multiple bug fixes and stability improvements.

### Update
- Offline support: the app can now run without an internet connection after you open it once (maps are not available offline yet)
- You can open your place on OpenStreetMaps
- You can now copy a place name and address by clicking on it.
- Small UI refresh and design improvements.


## 1.2

### Update
— Lists can now be exported not only in GPX format, but also in KMZ and CSV formats.
— The Share View design has been updated — it is now cleaner and more relaxed.
— Lists can now be password protected.

## 1.1

### Fixes
- Fixed navigation to 2GIS from the app.
- Resolved an issue where generating a QR code took too long when the list was large.
- Fixed vertical scrolling in nested lists.
- Fixed some design bugs.

### Features
- Added search inside lists when the number of items exceeds 10.
- Added Organic Maps as a navigation option.
